# Text Emotion Detection Using NLP

A web application that detects the **emotion in a given text** using Natural Language Processing (NLP) and machine learning. Built with **Streamlit**, **scikit-learn**, and **Python**.

---

## 🌐 Live Demo:
  https://textemotiondetectionusingnlp-cfi9kvwapghn3gd4drvksg.streamlit.app/

## 🚀 Features

- Detects emotions such as **anger, joy, sadness, fear, surprise, disgust** from text.
- Displays results with **corresponding emojis**.
- User-friendly **interactive Streamlit interface**.
- Quick and lightweight — no heavy models required.

---

## 🛠 Technologies Used

- **Python 3.13+**
- **Streamlit** for web app interface
- **scikit-learn** for machine learning
- **pandas** & **numpy** for data handling
- **joblib** for model serialization
- **Altair** for data visualization
